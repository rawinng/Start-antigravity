# Start-antigravity

This project documents the process of learning about Google Antigravity by building a simple web development project step-by-step.

## Requirements

*   **Single Page Website**: A responsive single-page application.
*   **Regex Learning Tool**: Includes a simple interface to input regular expressions and visualize matches against a set of sample paragraphs.
*   **Framework**: Next.js
*   **Styling**: Tailwind CSS
*   **Rendering**: Server-Side Rendering (SSR) support
*   **Deployment**: Deployed in containerized environment
*   **CI/CD**: GitHub Actions
